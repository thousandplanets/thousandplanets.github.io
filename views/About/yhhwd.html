    为什么，我如此沉醉银河护卫队？

    Valerian 5/11/2023 Movie

    心灵上的冲击与共鸣，不往往才是最打动人的嘛。

    一堆随性写的乱七八糟的情绪记录

    三天内完成了六刷1+2，二刷圣诞特辑，准备周末再去二刷一次3，无关它的槽点，优秀与否，纯粹是为陪伴我MCU最久最爱的老朋友们好好地道别。

    近十年来的回忆，明明不是我入漫威的初心，但却是我在MCU中最令我念念不忘的一群人。银护1和2在我心中同等的地位，是复联4也无法撼动的存在。Gunn对于群戏的掌握真的能秒杀太多商业导演了。

    # Vol.1

    优秀的导演即便是十八线的角色进行改编，同样可以细致入微，甚至是整个宇宙流水线系列的质量顶梁柱。

    120分钟内迅速交代五个新成员，交代力量宝石和罗南以及灭霸，七八十年代的金曲串烧每一首都符合角色当处的心境，复古感，笑点与正经随时都能无缝穿插，一帮失败者因各自的原因踏上了拯救银河的旅程。我要好好吹一吹Gunn对于群戏的掌控（重点放在Vol.2）没有伟光正，只是一帮有着各自苦衷和烦恼的怪胎，今天就是要打爆你罗南的故事。

    Gunn一定是偏爱火箭的，他一直喜欢用火箭的视角阐述着越脆弱的人越喜欢用别人讨厌自己来保护自己的方式，Vol2重点说。

    MCU在此之前的所有作品，无一少不了插科打诨也就是让角色时而严肃，但无论如何一定会诙谐化 幽默化，而Gunn的恶趣味完全符合这点，直接不装了，将这种风格贯彻到底，一帮不完美的失败者都有着各自的理由，但在人生的某一时刻，他们各自的决定自然地促成了团结在一起，看似降智的操作都能得到合理的解释符合每个角色的性格，这里是他们逐渐成为彼此最重要的家人 第一步。也是节奏最棒的一部。

    # Vol.2

    重点来啦。由于第一部过于精彩太多人吹捧因此贬低2，但我想重点笔墨一下，因为我非常非常喜欢Vol2，且绝对被很多人严重低估了。第二部很多人吐槽结构和叙事跳跃，压缩的太惨，导致什么线索都想塞 什么都讲的很散，但可能是我看了三四次的缘故了，每一条故事线和角色弧的把握，都在仅有篇幅内做到了漫威大框架下最完美的处理。且对角色的细腻拿捏的达到了争分夺秒的程度，绝对是和复联4平级的水平，且复联4是mcu最长的电影才能抗衡。

    银护2多线叙事的平行交叉有多么环环相扣？我一一阐述下。

    光是捋捋剧情上角色线就已经非常繁多了：

    奎尔与Ego、火箭与星爵与勇度，勇度与奎尔，Mantis与Drax，卡魔拉与奎尔，卡魔拉与星云。

    right，听着是很像报菜名，但是仔细沉下心来观赏下来，每一条线都是互相交集且有作用的，有角色成长和自我救赎的，甚至还铺垫了3。

    # 奎尔 EGO

    在最开头奎尔对女祭司说自己不在乎自己的父亲，他没有在母亲死前陪她，也从来没有回来过，他跟卡魔拉也一直固执着说不需要这个是真是假都无所谓的父亲，但是卡魔拉早就洞察了他的内心，他其实正是因为缺失，所以极度渴望，但是外表却做出拒绝的样子故作坚强这一点其实和勇度与火箭也对应上了。卡魔拉真的逐渐了解了奎尔，爱上了奎尔。这条剧情线促使了卡魔拉和星云的线，以及 火箭和勇度线以及火箭和奎尔的矛盾线。卡魔拉和奎尔云淡风轻的说，假如发现他是坏人了，我们一起杀了他就行，给了奎尔一个放下自己固执的台阶。后来和Ego迅速的构建了父子从小缺失的认同感，但他爱憎分明，得知真相能立马向自己父亲开枪。

    # 奎尔 卡魔拉

    卡魔拉和奎尔的谈话每一次都让人印象深刻，Vol1她从不表露自己内心，奎尔对音乐的爱让她产生了好奇，是什么东西即便豁出性命也要守护，是他和母亲的爱。在热闹的气氛下她在浪漫的七十年代金曲陶醉着经典的旋律，她看见了奎尔善良的内心与自己长期被灭霸压抑的善良吻合，最后试着放下一切成就 试着和他成为真正的家人。Vol2中，即便是奎尔被Ego洗脑过深产生些许争执的情况下，卡魔拉的语气仍然是带有平静与温和的，她逐渐卸下了用强大来伪装自己的脆弱，因为她了解奎尔真正的模样，他们互相改变了彼此。

    # 卡魔拉 星云

    在星云面前 嘴上说着彼此是死敌，但从未真的把刀刺向她。（这个打戏真的很细节）二者在缠打中也终于用言语道出了许多彼此的心声。

    甚至这时就已经暗示3中星云的归宿...

    另外 这种对称构图真的很戳我，这一次换我握住你的手，且绝不松开。

    # 勇度 火箭

    他俩都是最嘴上不饶人的豆腐心，他们都有着悲惨的身世，创造自己的人不在乎自己（这里也在铺垫3）自暴自弃干了许多坏事，用最恶劣的话和让别人讨厌的形象伪装掩饰自己的脆弱，每次和奎尔嘴炮互骂都不留余地，但得知有生命危险时，即便连续跳跃700次的扭曲也没有任何犹豫。

    # Drax Mantis

    Drax在Vol1是一个丧女丧妻失去理智的男人，他用怨气和愤怒伪装来麻痹自己，但被罗南狠狠的捶打清醒后，他虽然不懂很多冷笑话和直来直往，但他每次的出言不逊都是内心真实的想法，且总能打动到人，是最纯粹的人，他的自言自语创造了许多的笑点，也留下过不少泪点。

    Vol2 螳螂则刚好相反，她从初登场就在掩饰自己的内心，她一直是EGO的催眠"工具人"，第一次社交遇上直言直语的Drax，二者就进行了很好地互补，从最初的"你好丑，想到和你发生X关系我真的恶心"，再到"你可以不相信自己，我相信你" 他们的心路变化是最直白 笔墨最少，但足以印象深刻。

    # 交织

    星爵知道了自己真正的“父亲” 真正在乎自己的是谁，火箭也通过勇度意识到了自己最脆弱的地方，当他们刚刚深交之时，勇度的生命便为爱，迎来了终章。

    "有时候你穷极一生去寻找的东西，其实一直在你身边。"

    他们随心所欲的活着，偶尔拿”证据世界“当做外快，偶尔做回”小偷“的老行当，顽皮的时候相互吐槽，关键的时候相互鼓励，相互弥补 自我救赎，这就是该死的银河护卫队。

    星空下绽放的烟火是谱写给“英雄”的终章，是太空歌剧浪漫的间奏。

    # 圣诞特辑

    在经历多灾多难的第四阶段后，最佳的享受果然还是交由银护交上了答卷，

    节日

    会让我们回忆起曾经已故的人，会回忆已悄然褪去的时光。

    会让我们更珍惜现在的家人与爱人

    会让我们感谢身边的人

    会给我们一个更具仪式感向你所爱之人表达内心的机会

    是爱啊。

    是啊，唯独

    爱

    能穿越全宇宙。

    # 结

    其实还是省略了好多好多的细节没有概述，但是我感觉真的记录不完，这一次补完感觉真的是对自己喜欢了近十年的一群人做一个阶段性的道别，全程是有感而发的纪念。

    剧情服务角色，从而造就角色，而不是公式化的让你感动，Gunn一直能在有限的篇幅尽可能让他们展示更多的性格特点，且每条线都能得到自然地过渡绝不ooc。3的湫湫合理的展示Drax父亲的一面，星爵和卡魔拉的遗憾但也足够信服，星云的归宿源于铺垫，一切切种种故事的发展，都很好的和笑点结合自然发展，我真的太爱了....

    他的故事编排强大的地方在于从来不是强行编造和狗尾续貂，而是这些形形色色的人聚在一起，自然而然就会发展成这样了。

    虽然几乎没有出现family的台词，但每一次成长和互补的过程都在刻画，一帮曾经素未谋生，血缘 品种都没有关系的一帮人，纽带深深联系在一起成为彼此真正的家人。为其感到动容在正常不过，但我好像不只是动容这种程度。

    音乐 风格 情节 角色 场面，每一个都能让自己爱到这种程度的真的非常非常稀少，即便3我有一些不满的槽点，但他之前的优点依旧保留了下来，Gunn也依旧表明了对每个角色的尊重和热爱，光是这点我就足够满意了。

    黑寡妇：？ 漫威欠寡姐太多了

    即便之前没有相遇上又怎么样，正因为种种弯路和曲折，才造就现在的我们相遇，然后去创造更深的牵绊和回忆。银护的这份牵绊的感动，大概就是我最喜欢，也是最渴望的地方了吧。
